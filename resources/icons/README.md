# Ícones

Coloque aqui os ícones da aplicação:

- `icon.ico` - Ícone para Windows (256x256 pixels)
- `icon.icns` - Ícone para macOS (1024x1024 pixels)
- `icon.png` - Ícone para Linux (512x512 pixels)

## Como gerar

### Windows (.ico)
Use uma ferramenta como [icoconvert.com](https://icoconvert.com/) para converter o SVG para ICO.

### macOS (.icns)
Use o utilitário `iconutil` no macOS ou [icns.io](https://icns.io/).

### Linux (.png)
Use o arquivo SVG diretamente ou converta para PNG com [cloudconvert.com](https://cloudconvert.com/svg-to-png).

## Ícone temporário
O arquivo `icon.svg` pode ser usado como referência para gerar os ícones finais.
